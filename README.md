# My Solution for the Fawry Quantum Bookstore Challenge

## How to Run

1- Clone the repository:

   ```bash
   git clone https://github.com/hosahmed/Fawry-Quantum-Bookstore.git
   ```

2- Navigate to the project directory and compile the code:

   ```bash
   javac Main.java
   ```

3- Run the program:

   ```bash
   java Main
   ```
